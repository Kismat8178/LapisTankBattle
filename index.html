How to Play

Player 1 (Keyboard):

  WASD: Move Tank

  Arrow Keys (← → ↑ ↓): Aim Turret

  Spacebar: Shoot

  Left Shift: Use Active Skill


Player 2 (Keyboard):

  IJKL: Move Tank

  U / O: Rotate Turret

  H: Shoot

  Right Shift: Use Active Skill


Mobile Controls:

  Bottom-Left Joystick: Move Tank

  Bottom-Right Joystick: Aim Turret & Shoot

  Skill Button (Above Aim Joystick): Use Active Skill
